# CARA MENJALANKAN PROJECT
1. Menyalakan XAMPP
  * Buka XAMPP Control Panel
  * Jalankan Apache dan MySQL
  * Pastikan status keduanya RUNNING
2. Membuat Database
  * Buka PhpMyadmin
  * Buat database dengan nama node_c
  * Import atau jalankan query untuk membuat tabel category dan product
3. Clone Repository dari GitHub
  * Buka terminal (CMD / Terminal VS Code)
  * Jalankan perintah: https://github.com/RamadhaniUcup/nodejs-crud-restapi.git
  * cd nodejs-crud-restapi
4. Selanjutnya Menjalankan Server
  * Buka terminal masuk terminal ke CMD jalankan aplikasi dengan npm run dev
5. Testing API Menggunakan Postman
  * Buka Postman
  * Contoh endpoint: GET data product
  * http://localhost:3000/product
